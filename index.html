FAQ

Common questions.

Do I need a Twitter developer account? +
Yes. You need a Twitter Developer account with Basic access (~$100/month from Twitter). PumpSniper uses the official Twitter API v2 to scan tweets in real-time.
Is my wallet private key safe? +
PumpSniper is non-custodial. Your private key is stored only on your own machine in your .env file and never sent to our servers. We recommend using a dedicated wallet with limited funds.
How fast is "fast"? +
Detection-to-launch averages 0.3 seconds from when we detect the tweet spike. Elite users get priority queue access for fastest possible execution. Actual blockchain confirmation time varies by Solana network congestion.
Can I try it before buying? +
All plans include a Test Mode that simulates launches without spending real SOL. You can verify detection speed and setup before going live.
What blockchains are supported? +
Currently Solana only, via Pump.fun. We're working on adding support for Base and Ethereum L2s in a future update — included free for all existing customers.
